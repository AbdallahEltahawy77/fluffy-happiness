# 🌟 Glow House Sales Performance & Forecasting Dashboard

![Dashboard Preview](رابط_صورة_الداشبورد_هنا)

## 📝 Overview
An interactive data analytics solution developed for **Glow House (Saudi Arabia)** to track sales performance across different dimensions and provide data-driven forecasts for the upcoming year.

## 🚀 Key Features
* **Total Revenue Tracking:** Real-time monitoring of revenue vs targets.
* **Performance Comparison:** MOM (Month-over-Month) analysis to identify growth trends.
* **Top Performers:** Insights into the Top 10 Products and Top 10 Stores.
* **Segmentation:** Breakdown of sales by Item Class and Transaction Type (Cash, Insurance, Online, Wasfaty).
* **Predictive Analytics:** Forecasting models to anticipate future sales trends for 2026.

## 🛠️ Tech Stack
* **Tool:** Power BI / Excel (حسب الأداة اللي استخدمتها)
* **Data Source:** SQL Database / Excel Sheets
* **Analysis:** DAX for complex measures and forecasting.

## 📊 Business Insights (Quick Hits)
* **Cash Sales dominance:** ZMDN class accounts for over 69% of sales.
* **Digital Presence:** Online sales contribute 5.3% – a growth opportunity identified.
* **Top Product:** "Free Style Libre-2" is the hero product with $2.77M in revenue.
